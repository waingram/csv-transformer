> Transforms CSV to JSON and RDF.


## Getting Started

Install the module with: `npm install csvw`

```js
var csvld = require('csvw');
csvw.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g csvw
$ csvw --help
$ csvw --version
```
